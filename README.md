# Studio 27 Salon Website

Full-stack MERN project for a premium salon booking website.

## Features
- Salon service showcase
- Appointment booking system
- Contact form
- Slot availability checking
- Responsive UI

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS

## Project Structure
frontend/ → React application
backend/ → Express API and MongoDB

## Installation
1. Clone repo
2. Install dependencies
3. Run backend and frontend
